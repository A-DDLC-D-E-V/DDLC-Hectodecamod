THIS MOD IS TO BE PLAYED AFTER THE PREVIOUS MOD (DDLC Hectomod) AS IT IS A SEQUEL AND CONTAINS REFERENCES TO THE PREVIOUS MODS: https://github.com/A-DDLC-D-E-V/DDLC-Hectomod

Download and extract the included mod's ZIP. Get the official DDLC game from ddlc.moe. Go into the game folder of the official game, and add the official game's images.rpa, audio.rpa, and fonts.rpa into the mod's game folder, or it will throw an error. Run the EXE (or SH for Linux) and enjoy! There may be a default username and/or a messed up start screen, which are remnants from my testing (IDK how to remove them).
